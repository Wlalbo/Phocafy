# Phocafy

## What is this for?

Application that gives you seal facts whenever you type in some sort of keyword.
For now it is quite small and only has three seal facts two of which are quite banal and simple but it'll have more in the future.

## How do you run it?

In your terminal:

`gcc main.c -o phocafy`
`./phocafy`

Done. Enjoy.

## What language is it in?

It is in C. It is mostly used to test my C skills as I am learning.
